Lighting Software

  • Avolites Titan - PC version of the Titan software which runs on the Avolites desk. Free to use until version 11.4 although it spits out random data once in a while. Requires a dongle from version 12. ⚠ not free.
  • Chamsys MagicQ - MagiQ is a professional lighting software providing up to 64 Artnet universes for free. It includes powerful features like 2D/3D pixel mapping, effect generator, media server and visualizer.
  • Daslight - DMX lighting software for PC and Mac.
  • Dot2 software - Free, offline software for the Dot2 consoles. Includes Dot2 onPC and Dot2 3D.
  • EOS Family - Software for ETC’s EOS-family consoles. ⚠ freemium.
  • Freestyler - Free lightning control software
  • GrandMA2 software - Free, offline software for the GrandMA2 lighting console. Includes GrandMA2 OnPC and MA3D. Does not output DMX for free. ⚠ freemium.
  • GrandMA3 software - Free, offline software for the new GrandMA3 lighting console. Includes GrandMA3 OnPC. Does not output DMX for free. ⚠ freemium.
  • Hog 4 software - Free, offline software for the Hog 4 lighting console. Includes Hog 4 pc. Does not output DMX for free. ⚠ freemium.
  • LightFactory - PC Lighting Control software witch also is the backend of the NEO Console. ⚠ not free.
  • ONYX - Modern lighting control system with alot of features.
  • QLC+ - QLC+ is a free and cross-platform software to control DMX or analog lighting systems like moving heads, dimmers, scanners etc. Source Code ✓ open-source.
  • xLights - Meant for home Christmas light shows but is a great way to sync lights to music on a budget. Source Code ✓ open-source.

Visualizers

  • Capture - Documentation and visualisation software. Capture is a native macOS and Windows application that lets you work with lighting, video, laser, moving scenery and water effects. Does have a free Student Edition. ⚠ freemium.
  • Depence² - Depence² is a unique application, which combines the design and visualization process with the potential to control the real show. ⚠ not free.
  • Ma3D - Free lighting visualizer for GrandMA consoles/software.
  • wysiwyg - wysiwyg is an all-in-one lighting design software with fully integrated CAD, plots, data, visualization and virtual show control. ⚠ not free.

Audio Software

  • Audacity - Audio file editor and analysis tool, great for cropping down sound effects and music tracks. ✓ open-source.
  • Ardour - A full realtime audio recording and editing suite. Source Code ✓ open-source.
  • da-Share MultiPlay - MultiPlay is a Windows based program designed to play audio cues for theatre or corporate use. It is free to use.
  • Linux Show Player - Free cue player designed for stage productions. Source Code ✓ open-source.
  • QLab - QLab is sound, video, and lighting control for macOS. ⚠ freemium.
  • REAPER - REAPER is a complete digital audio production application for computers, offering a full multitrack audio and MIDI recording, editing, processing, mixing and mastering toolset. ⚠ freemium.

Audio tools

  • ARTA - A collection of programs for audio measurements and analysis.
  • REW - REW is free room acoustics analysis software for measuring and analysing room and loudspeaker responses. Free SMAART alternative.

Tools

^ back to top ^

  • Livescript - Insert a musical/theatre-script from Google Docs and use this for a live “autocue” and scroller with everyone on the site. ✓ open-source.
  • StageClock - StageClock is a Java based graphical timekeeping application. It outputs a clean feed from an attached monitor, and has time/date, timer, and show information. ⚠ freemium.

CADs

  • AutoCAD - AutoCAD® is computer-aided design (CAD) software that architects, engineers, and construction professionals rely on to create precise 2D and 3D drawings. ⚠ not free.
  • Vectorworks - Vectorworks is a versatile, on-premise application that provides extensive 2D drafting, 3D modeling, BIM and rendering capabilities for your architectural and landscape design needs. Also had a special entertainment version: Vectorworks Spotlight. ⚠ not free.

Timecode

Timecode is a sequence of numeric codes generated at regular intervals by a timing synchronization system. Timecode is used in video production, show control and other applications which require temporal coordination or logging of recording or actions.

  • El-Tee-See - LTC wav file generator.
  • TCgenerator - Reaper time marker to GrandMA2 TC Generator. ✓ open-source.
  • Timecode - ArtNet and LTC timecode generator, made with Java. ✓ open-source.
  • Time MIDI sender - A free tool for sending time data to DasLight (via MIDI). ✓ open-source.
  • TimelordMTC - Advanced timecode management tool. ⚠ not free.
  • M2Q - Python application capable to trigger Chamsys Consoles/Software via MIDI for advanced synchronized shows. ✓ open-source.

Monitoring

  • AES67 WebMeter - Server that receives an AES67 audio stream and presents a webpage with meters. ✓ open-source.
  • meters.lv2 - meters.lv2 is a collection of audio-level meters with GUI in LV2 plugin format. ✓ open-source.
  • Open Sound Meter - Clean, open-source sound meter. Supports RTA, magnitude, phase and impulse response, coherence, group delay and more. ✓ open-source.
